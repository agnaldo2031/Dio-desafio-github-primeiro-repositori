 	- SHA1

- A sigla SHA significa Secure Hash Algorithm
(Algoritimo de hash Seguro), é um conjunto de
funções hash criptográficas projetadas pela
NSA (Agência de Segurança Nacional dos EUA).

- A encriptação gera conjunto de characteres 
indentificador de 40 dígitos

 ex:

echo "ola mundo" | openssl sha1
(vai gerar um linha de código desta string)


	- Objetos fundamentais / internos do Git


	- BLOBS

- Contém metadados ( que são o tamanho do arquivo 
tipo do objeto etc...

- Não guarda o nome do ar

- Bloco basico da composição

Exemplo:

- echo "conteudo" | git hash-object --stdin

- echo -e "blob 9\0conteudo" | openssl sha1

	- TREES

-  São arvores que amazena BLOBS

-  Tem os nomes do arquivos

- Contém metadados

- Contém o sha1 dos arquivos

- Guarda os nomes dos arquivos

- E respondsavel por motar toda a estrutura de onde estão 
localizados esses aqruivos 
 
- Pode pontar para BOLBs que são "arquivos" ou outra 
arvores.

- As arvores a ponta para os sha1 dos BLOBs que tem ali 
sha1 das arvores

	- COMMITS

- tambem tem sha1 que modifica quando ha alteração nos
BLOs TREE E Commit parente apontar ou relacionados assim
toda mudança na estrutura vai gerar um novo sha1
